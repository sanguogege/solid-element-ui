## Popover 气泡卡片

高度封装的浮层组件，支持点击触发展示富文本内容。

### 导入
```tsx
import { Popover } from "solid-element-ui";
```

### 使用示例

```tsx
<Popover 
  trigger={<span class="text-blue-500 cursor-help">查看帮助</span>}
>
  <p class="text-sm">这是气泡卡片的内容区域。</p>
</Popover>
```

### API 参考

| 属性 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| **trigger** | `JSX.Element` | - | **必填**。触发元素。 |
| **showArrow** | `boolean` | `false` | 是否显示装饰箭头。 |
| **placement** | `Placement` | `'bottom'` | 弹出位置。 |
| **modal** | `boolean` | `false` | 是否开启模态拦截。 |

---

### 注意事项
由于 Kobalte 的 `Trigger` 默认渲染为 `button` 标签，建议 `trigger` 属性传入文本或图标。如果需要传入复杂的自定义组件且不希望被包裹在 `button` 中，可以使用 `as` 属性进行覆盖。
