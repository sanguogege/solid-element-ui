## RatingGroup 评分

高度封装的评分组件，内置 Lucide 图标并支持动态高亮与 Form 绑定。

### 导入
```tsx
import { RatingGroup } from "solid-element-ui";
```

### 使用示例

```tsx
<div class="flex flex-col gap-6">
  {/* 基础用法 */}
  <RatingGroup label="服务评分" defaultValue={3} />

  {/* 蓝色风格 + 10星模式 */}
  <RatingGroup 
    label="难度系数" 
    count={10} 
    color="blue" 
    size="sm" 
  />

  {/* 只读模式 */}
  <RatingGroup value={4.5} readOnly label="综合评分" />
</div>
```

### API 参考

| 属性 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| **count** | `number` | `5` | 总分值（星星数量）。 |
| **label** | `string` | - | 顶部的描述文字。 |
| **color** | `'yellow' \| 'blue' \| 'red'` | `'yellow'` | 高亮颜色。 |
| **size** | `'sm' \| 'md' \| 'lg'` | `'md'` | 图标尺寸。 |
| **allowHalf** | `boolean` | `false` | 是否允许半星（需配合特定的样式调整）。 |
| **disabled** | `boolean` | `false` | 禁用交互。 |
| **readOnly** | `boolean` | `false` | 只读模式，仅用于展示。 |

---

### 特性说明
1. **自动高亮**：利用 Kobalte 内置的 `data-[highlighted]` 属性，结合 Tailwind 变体自动切换 `fill` 和 `color`。
2. **键盘支持**：支持 `ArrowLeft` 和 `ArrowRight` 进行精准分值调节，符合 WAI-ARIA 标准。
3. **隐身输入框**：内置 `HiddenInput`，可无缝配合原生 `<form>` 提交。
