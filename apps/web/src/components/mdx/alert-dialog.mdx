## Alert Dialog 警告对话框

### 导入

```tsx
import { AlertDialog } from "solid-element-ui";
```

### 使用示例

```tsx
<AlertDialog 
  title="确认删除？"
  description="此操作不可逆。"
  trigger={<button className="text-red-600">删除</button>}
  action={<button className="bg-red-600 text-white px-4 py-2 rounded">确定</button>}
/>
```

### Tailwind v4 动画

```css
@import "tailwindcss";

@theme {
  @keyframes fade-in { from { opacity: 0; } to { opacity: 1; } }
}

[data-kb-overlay] { animation: fade-in 0.2s; }
[data-kb-content] { animation: fade-in 0.2s; }
```
