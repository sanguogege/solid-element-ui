## Breadcrumbs 面包屑

显示当前页面在层级结构中的位置，并允许用户向上回溯。

### 导入

```tsx
import { Breadcrumbs } from "solid-element-ui";
```

### 使用示例

在 MDX 中定义导航路径并渲染：

```tsx
export const navItems = [
  { title: "首页", href: "/" },
  { title: "组件库", href: "/components" },
  { title: "面包屑", current: true }
];

<div className="p-4 bg-white dark:bg-zinc-950 border rounded-lg">
  <Breadcrumbs items={navItems} />
</div>
```

---

### API 参考

| 属性 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| **items** | `BreadcrumbItem[]` | `[]` | **必填**。面包屑的数据数组。 |
| **separatorIcon** | `JSX.Element` | `<ChevronRight />` | 自定义分隔符图标。 |
| **class** | `string` | - | 覆盖根容器的 Tailwind 类名。 |

### BreadcrumbItem 接口

| 字段 | 类型 | 说明 |
| :--- | :--- | :--- |
| **title** | `JSX.Element \| string` | 导航显示的文字或图标。 |
| **href** | `string` | 跳转链接。如果未提供，则仅展示文字。 |
| **current** | `boolean` | 是否为当前页面（会应用 `data-current` 样式）。 |
| **disabled** | `boolean` | 是否禁用该链接。 |

---

### Tailwind v4 适配

Breadcrumbs 主要使用 `data-current` 状态。在 Tailwind v4 中，你可以通过 CSS 变量微调链接样式：

```css
@import "tailwindcss";

@theme {
  /* 定义面包屑专属的颜色变量（可选） */
}

/* 针对当前页面的特殊动画或样式 */
[data-kb-breadcrumbs-link][data-current] {
  color: var(--color-zinc-900);
  text-decoration: none;
  cursor: default;
}

/* 列表容器响应式处理 */
ol {
  display: flex;
  flex-wrap: wrap;
}
```
