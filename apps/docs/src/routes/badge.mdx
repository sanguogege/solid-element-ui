## Badge 徽章

基于 Kobalte 的语义化徽章组件，用于展示状态或标签。

### 导入

```tsx
import { Badge } from "solid-element-ui";
```

### 基础用法

在 MDX 中渲染基于 Kobalte 的 Badge 实例：

```tsx
<div className="flex flex-wrap gap-2 p-4">
    <Badge variant="default">Beta</Badge>
    <Badge variant="success">已上线</Badge>
    <Badge variant="danger">已下线</Badge>
    <Badge variant="outline">v1.0.0</Badge>
</div>
```

---

### API 参考

| 属性        | 类型                                                             | 默认值      | 说明                                    |
| :---------- | :--------------------------------------------------------------- | :---------- | :-------------------------------------- |
| **variant** | `'default' \| 'secondary' \| 'outline' \| 'success' \| 'danger'` | `'default'` | 预设的视觉风格。                        |
| **class**   | `string`                                                         | -           | 覆盖样式的 Tailwind 类名。              |
| **as**      | `string \| Component`                                            | `'div'`     | 修改渲染的 HTML 标签（如改为 'span'）。 |

---

### Tailwind v4 适配

在 Tailwind v4 环境下，如果你需要徽章在数值更新时有一个微小的缩放动画，可以在 `index.css` 中添加：

```css
@import "tailwindcss";

@theme {
    @keyframes badge-pop {
        0% {
            transform: scale(1);
        }
        50% {
            transform: scale(1.1);
        }
        100% {
            transform: scale(1);
        }
    }
}

/* 当 Badge 内容改变时可手动触发或应用此样式 */
.badge-update {
    animation: badge-pop 0.2s ease-out;
}
```
