


## Breadcrumbs 面包屑

显示当前页面在层级结构中的位置，并允许用户向上回溯。

import {DemoCode} from "~/components/breadcrumbs"

<DemoCode />


### 基础用法

在 MDX 中定义导航路径并渲染：

```tsx
import { Breadcrumbs } from "solid-element-ui";


export const navItems = [
    { title: "首页", href: "/" },
    { title: "组件库", href: "/components" },
    { title: "面包屑", current: true },
];

<div class="p-4 bg-white dark:bg-zinc-950 border rounded-lg">
    <Breadcrumbs items={navItems} />
</div>;
```

---

### API 参考

| 属性              | 类型               | 默认值             | 说明                         |
| :---------------- | :----------------- | :----------------- | :--------------------------- |
| **items**         | `BreadcrumbItem[]` | `[]`               | **必填**。面包屑的数据数组。 |
| **separatorIcon** | `JSX.Element`      | `<ChevronRight />` | 自定义分隔符图标。           |
| **class**         | `string`           | -                  | 覆盖根容器的 Tailwind 类名。 |

### BreadcrumbItem 接口

| 字段         | 类型                    | 说明                                           |
| :----------- | :---------------------- | :--------------------------------------------- |
| **title**    | `JSX.Element \| string` | 导航显示的文字或图标。                         |
| **href**     | `string`                | 跳转链接。如果未提供，则仅展示文字。           |
| **current**  | `boolean`               | 是否为当前页面（会应用 `data-current` 样式）。 |
| **disabled** | `boolean`               | 是否禁用该链接。                               |
