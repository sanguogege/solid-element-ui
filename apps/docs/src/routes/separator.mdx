## Separator 分隔线

用于视觉上分隔内容区域，支持水平和垂直两种方向。

### 导入

```tsx
import { Separator } from "solid-element-ui";
```

### 基础用法

在菜单或卡片布局中使用：

```tsx
<div className="flex flex-col gap-2 p-4">
    <h4 className="font-medium text-sm">通知设置</h4>
    <p className="text-xs text-slate-500">管理您的应用推送偏好。</p>

    <Separator className="my-2" />

    <div className="flex h-5 items-center space-x-4 text-sm">
        <div>个人资料</div>
        <Separator orientation="vertical" />
        <div>账户</div>
        <Separator orientation="vertical" />
        <div>安全</div>
    </div>
</div>
```

---

### API 参考

| 属性            | 类型                                | 默认值         | 说明                   |
| :-------------- | :---------------------------------- | :------------- | :--------------------- |
| **orientation** | `'horizontal' \| 'vertical'`        | `'horizontal'` | 分隔线的方向。         |
| **thickness**   | `'thin' \| 'medium' \| 'thick'`     | `'thin'`       | 分隔线的粗细。         |
| **variant**     | `'default' \| 'muted' \| 'primary'` | `'default'`    | 视觉风格变体。         |
| **class**       | `string`                            | -              | 附加的 Tailwind 类名。 |

---

### 设计规范

1. **间距**：默认水平方向带有 `my-4`，垂直方向带有 `mx-4`。如果需要紧凑布局，可通过 `class="m-0"` 覆盖。
2. **语义化**：基于 Kobalte 构建，会自动处理 `role="separator"`，并根据 `orientation` 设置正确的 `aria-orientation`。
3. **垂直布局注意**：在使用 `orientation="vertical"` 时，请确保父容器具有明确的高度或使用 `items-stretch`。
