### API 参考

| 属性          | 类型          | 默认值      | 说明                         |
| :------------ | :------------ | :---------- | :--------------------------- |
| **variant**   | `Enum`        | `'default'` | 样式变体。                   |
| **size**      | `Enum`        | `'default'` | 按钮尺寸。                   |
| **loading**   | `boolean`     | `false`     | 是否显示加载状态并禁用交互。 |
| **leftIcon**  | `JSX.Element` | -           | 按钮文本左侧的图标。         |
| **rightIcon** | `JSX.Element` | -           | 按钮文本右侧的图标。         |
