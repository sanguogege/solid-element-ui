## TimeField 时间输入框

允许用户输入特定时间（小时、分钟、秒、AM/PM），支持键盘操作和分段验证。

### 导入

```tsx
import { TimeField } from "solid-element-ui";
import { parseTime } from "@internationalized/date";
```

### 基础用法

```tsx
const [value, setValue] = createSignal(parseTime("14:30"));

<TimeField
    label="会议时间"
    value={value()}
    onChange={setValue}
    description="请选择下午的会议时段。"
/>;
```

---

### API 参考

| 属性             | 类型                             | 默认值     | 说明                                         |
| :--------------- | :------------------------------- | :--------- | :------------------------------------------- |
| **label**        | `string`                         | -          | 顶部的标签文字。                             |
| **value**        | `Time`                           | -          | `@internationalized/date` 提供的 Time 对象。 |
| **hourCycle**    | `12 \| 24`                       | -          | 强制使用 12 小时制或 24 小时制。             |
| **granularity**  | `'hour' \| 'minute' \| 'second'` | `'minute'` | 时间精度。                                   |
| **size**         | `'sm' \| 'md' \| 'lg'`           | `'md'`     | 控件尺寸。                                   |
| **errorMessage** | `string`                         | -          | 错误提示，传入后会自动变红。                 |

---

### 关键细节

1. **分段渲染**：组件内部使用了 `state.segments()` 遍历，这能确保小时、分钟、分隔符（冒号）都被渲染为独立的、可单独聚焦的元素。
2. **本地化支持**：Kobalte 会根据用户的浏览器区域设置自动处理 `AM/PM` 的显示位置。
3. **数据类型**：该组件通常需要配合 `@internationalized/date` 库使用，以确保时间计算的准确性。
