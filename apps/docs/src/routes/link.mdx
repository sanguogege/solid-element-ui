# Link 链接

用于在不同页面之间导航或跳转到外部资源的交互组件。基于 Kobalte 的 Link 核心构建，支持多态渲染和增强的无障碍语义。

## 引入

import { Link } from "solid-element-ui";

## 代码演示

### 1. 基础用法

默认渲染为具有链接样式的锚点元素。

<Link href="https://solidjs.com" target="_blank">
    访问 SolidJS 官网
</Link>

### 2. 禁用状态

禁用后的链接将无法点击，并自动从 Tab 键导航流中移除。

<Link href="/dashboard" disabled class="text-zinc-400">
    个人中心 (暂未开放)
</Link>

### 3. 多态渲染 (As)

结合前端路由库（如 `@solidjs/router`）使用，保持 UI 样式的一致性。

<Link as={A} href="/about" class="font-medium text-blue-600">
    关于我们
</Link>

## API 参考

### Link

| 属性     | 说明                 | 类型           | 默认值 |
| :------- | :------------------- | :------------- | :----- |
| href     | 跳转地址             | string         | -      |
| disabled | 是否禁用             | boolean        | false  |
| as       | 底层渲染的元素或组件 | ValidComponent | 'a'    |
| class    | 自定义样式类名       | string         | -      |

## 无障碍 (Accessibility)

-   语义化: 默认使用 `<a>` 标签。当使用 `disabled` 属性时，组件会移除 `href` 并添加 `aria-disabled="true"` 以确保辅助技术能正确识别。
-   键盘导航: 支持通过 `Tab` 键聚焦，并按 `Enter` 键激活。
-   安全性: 对于外部链接，建议手动添加 `rel="noreferrer noopener"`，组件会自动传递这些原生属性。
