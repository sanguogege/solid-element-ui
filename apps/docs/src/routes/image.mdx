## Image 图片

基于 Kobalte Image 封装的增强图片组件，支持在图片加载失败或地址为空时自动显示占位内容。

### 导入

```tsx
import { Image } from "solid-element-ui";
```

### 基础用法

```tsx
<div class="flex gap-4">
    {/* 正常显示 */}
    <Image
        class="w-20 h-20"
        src="[https://github.com/kobaltedev.png](https://github.com/kobaltedev.png)"
        radius="full"
    />

    {/* 加载失败显示 Fallback (默认取 alt 前两个字母) */}
    <Image class="w-20 h-20" src="invalid.jpg" alt="Gemini" radius="md" />

    {/* 自定义 Fallback 内容 */}
    <Image
        class="w-20 h-20"
        src="invalid.jpg"
        fallback={<div class="text-xs">Error</div>}
    />
</div>
```

### API 参考

| 属性                      | 类型                                       | 默认值    | 说明                                         |
| :------------------------ | :----------------------------------------- | :-------- | :------------------------------------------- |
| **src**                   | `string`                                   | -         | 图片资源路径。                               |
| **alt**                   | `string`                                   | -         | 图片描述，同时作为默认 Fallback 的文字来源。 |
| **fallback**              | `JSX.Element`                              | -         | 加载失败或加载中时显示的自定义内容。         |
| **radius**                | `'none' \| 'sm' \| 'md' \| 'lg' \| 'full'` | `'none'`  | 容器圆角设置。                               |
| **loadingStatusStrategy** | `'eager' \| 'lazy'`                        | `'eager'` | Kobalte 原生属性：决定何时渲染 fallback。    |

---

### 样式说明 (Tailwind v4)

组件使用了 `shrink-0` 确保在 Flex 容器中不会被挤压，`aspect-square` 确保默认占位区域为正方形。

```css
/* 如果需要图片加载时的淡入效果 */
[data-kb-image-img] {
    animation: img-fade 0.3s ease-in;
}

@keyframes img-fade {
    from {
        opacity: 0;
    }
    to {
        opacity: 1;
    }
}
```
