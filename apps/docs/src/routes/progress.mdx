# Progress 进度条

展示一个长时间运行操作的完成进度。基于 Kobalte 的 Progress 核心构建，提供语义化的状态更新，确保屏幕阅读器用户能够实时感知进度变化。

---

## 引入

import { Progress } from "solid-element-ui";

---

## 代码演示

### 1. 基础用法

展示一个简单的百分比进度。

<Progress value={70}>
    <div class="flex justify-between mb-1 text-sm">
        <Progress.Label>正在下载资源...</Progress.Label>
        <Progress.ValueLabel />
    </div>
    <Progress.Track class="h-2 bg-zinc-100 rounded-full overflow-hidden">
        <Progress.Indicator class="h-full bg-blue-600 transition-all duration-300" />
    </Progress.Track>
</Progress>

### 2. 不确定状态 (Indeterminate)

当无法估算具体进度（例如正在连接服务器）时，不传递 value 属性，通过 CSS 实现循环动画。

<Progress>
    <Progress.Track class="h-1 bg-zinc-100 overflow-hidden">
        <Progress.Indicator class="w-1/3 h-full bg-zinc-900 animate-progress-indeterminate" />
    </Progress.Track>
</Progress>

## API 参考

### Progress (Root)

| 属性          | 说明               | 类型                                                            | 默认值 |
| :------------ | :----------------- | :-------------------------------------------------------------- | :----- |
| value         | 当前进度值         | number                                                          | -      |
| minValue      | 最小值             | number                                                          | 0      |
| maxValue      | 最大值             | number                                                          | 100    |
| getValueLabel | 自定义数值文本格式 | (params: { value: number, min: number, max: number }) => string | -      |

### Progress.Track

| 属性  | 说明                 | 类型   | 默认值 |
| :---- | :------------------- | :----- | :----- |
| class | 进度条轨道的样式类名 | string | -      |

### Progress.Indicator

| 属性  | 说明                             | 类型   | 默认值 |
| :---- | :------------------------------- | :----- | :----- |
| class | 进度指示条（填充部分）的样式类名 | string | -      |

## 无障碍 (Accessibility)

-   **角色定义**：自动分配 `role="progressbar"`。
-   **状态更新**：自动管理并更新 `aria-valuenow`、`aria-valuemin` 和 `aria-valuemax` 属性。
-   **语义关联**：`Progress.Label` 自动通过 `aria-labelledby` 与进度条组件关联。
-   **不确定状态**：当 `value` 为空或未定义时，自动移除 `aria-valuenow` 属性，以符合 W3C 标准中关于不确定进度的定义。
