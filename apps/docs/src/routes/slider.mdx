## Slider 滑块

允许用户从一系列值中做出选择，支持单滑块和双范围滑块模式。

### 导入

```tsx
import { Slider } from "solid-element-ui";
```

### 基础用法

```tsx
// 基础使用
<Slider label="音量" defaultValue={[50]} showValue />

// 范围选择 (多滑块)
<Slider
  label="价格区间"
  defaultValue={[20, 80]}
  variant="success"
  showValue
/>
```

---

### API 参考

| 属性             | 类型                                 | 默认值      | 说明                           |
| :--------------- | :----------------------------------- | :---------- | :----------------------------- |
| **defaultValue** | `number[]`                           | `[0]`       | 初始值，数组长度决定滑块数量。 |
| **value**        | `number[]`                           | -           | 受控值。                       |
| **label**        | `string`                             | -           | 滑块左上角的标签文字。         |
| **showValue**    | `boolean`                            | `false`     | 是否在右上角显示当前数值。     |
| **variant**      | `'default' \| 'danger' \| 'success'` | `'default'` | 配色主题。                     |
| **size**         | `'sm' \| 'md' \| 'lg'`               | `'md'`      | 轨道和滑块的大小规格。         |
| **step**         | `number`                             | `1`         | 步长。                         |
| **min / max**    | `number`                             | `0 / 100`   | 最小值和最大值。               |

---

### 交互说明

1. **多滑块渲染**：通过 `For` 循环动态渲染 `Thumb`。当 `value` 或 `defaultValue` 长度大于 1 时，组件自动变为范围选择模式。
2. **无障碍**：内部自动关联 `KSlider.Input`，支持键盘方向键精确调整。
3. **触控支持**：集成 Kobalte 的 `touch-none` 逻辑，确保在移动端滑动顺畅。
