## Toast 消息提示

在屏幕角落显示的临时通知消息。


import {DemoCode} from "~/components/toast"

<DemoCode />


### 导入与配置

首先，你需要在应用的根节点放置 `Toaster` 容器：

```tsx
import { ToastProvider } from "solid-element-ui";

function App() {
    return (
        <>
            <Router
                root={(props) => (
                    <>
                        <ToastProvider />
                        ... routers
                    </>
                )}
            >
                <FileRoutes />
            </Router>
        </>
    );
}
```

### 基础用法

```tsx
import { showToast } from "solid-element-ui";

<Button
    variant="default"
    onClick={() =>
        showToast({
            variant: "success",
            title: "操作成功",
            description: "数据已同步至云端",
        })
    }
>
    确定
</Button>;
```

---

### API 参考

| 属性            | 类型                                          | 默认值   | 说明                       |
| :-------------- | :-------------------------------------------- | :------- | :------------------------- |
| **variant**     | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | 提示类型。                 |
| **title**       | `string`                                      | -        | 粗体标题。                 |
| **description** | `string`                                      | -        | 辅助详细信息。             |
| **duration**    | `number`                                      | `5000`   | 自动关闭时间（毫秒）。     |
| **persistent**  | `boolean`                                     | `false`  | 为 `true` 时不会自动关闭。 |

### 动画配置 (Tailwind v4)

```css
@theme {
    @keyframes toast-in {
        from {
            transform: translateX(100%);
        }
        to {
            transform: translateX(0);
        }
    }
}
```
