### 基础用法

```tsx
import { AlertDialog, Button } from "solid-element-ui";


<div class="p-4 bg-white dark:bg-zinc-950 border rounded-lg space-x-2 space-y-2">
    <AlertDialog
        title="确认删除？"
        description="此操作不可逆。"
        trigger={<Button>点击弹出</Button>}
    />

    <AlertDialog
        title="删除提醒"
        onConfirm={() => alert("删除提醒")}
        trigger={<button>点击删除</button>}
    />

    <AlertDialog
        title="确认删除？"
        description="此操作不可逆。"
        trigger={<Button color="warning">自定义action</Button>}
        action={
            <Button onclick={() => alert("确认删除？")} color="danger">
                确定
            </Button>
        }
    />
</div>
```

## API 参考

### Accordion Props

| 参数            | 类型              | 默认值  | 说明                             |
| :-------------- | :---------------- | :------ | :------------------------------- |
| **items**       | `AccordionItem[]` | `[]`    | **必填**。渲染面板的数据源。     |
| **multiple**    | `boolean`         | `false` | 是否允许多个面板同时展开。       |
| **collapsible** | `boolean`         | `false` | 是否允许关闭当前已展开的面板。   |
| **class**       | `string`          | -       | 覆盖根容器样式的 Tailwind 类名。 |

### AccordionItem 接口

| 字段         | 类型                    | 说明           |
| :----------- | :---------------------- | :------------- |
| **value**    | `string`                | 唯一标识符。   |
| **title**    | `JSX.Element \| string` | 标题内容。     |
| **content**  | `JSX.Element \| string` | 展开后的内容。 |
| **disabled** | `boolean`               | 是否禁用。     |
