## ToggleGroup 切换分组

一组可以被选中或取消选中的双态按钮集合，支持单选或多选。

### 导入

```tsx
import { ToggleGroup } from "solid-element-ui";
import { AlignLeft, AlignCenter, AlignRight } from "lucide-solid";
```

### 基础用法

```tsx
// 单选模式 (类似 SegmentedControl 但更具按钮感)
<ToggleGroup
  type="single"
  defaultValue="left"
  options={[
    { label: <AlignLeft size={18} />, value: "left" },
    { label: <AlignCenter size={18} />, value: "center" },
    { label: <AlignRight size={18} />, value: "right" },
  ]}
/>

// 多选模式
<ToggleGroup
  type="multiple"
  options={[
    { label: "加粗", value: "bold" },
    { label: "倾斜", value: "italic" },
    { label: "下划线", value: "underline" },
  ]}
/>
```

---

### API 参考

| 属性         | 类型                     | 默认值      | 说明                         |
| :----------- | :----------------------- | :---------- | :--------------------------- |
| **options**  | `Option[]`               | `[]`        | 选项列表。`label` 支持 JSX。 |
| **type**     | `'single' \| 'multiple'` | `'single'`  | 决定是单选还是多选。         |
| **variant**  | `'default' \| 'outline'` | `'default'` | 视觉风格变体。               |
| **size**     | `'sm' \| 'md' \| 'lg'`   | `'md'`      | 按钮尺寸。                   |
| **value**    | `string \| string[]`     | -           | 受控值（取决于 `type`）。    |
| **onChange** | `(val: any) => void`     | -           | 状态改变时的回调。           |
| **disabled** | `boolean`                | `false`     | 禁用整个分组。               |

---

### 关键细节

1. **多选逻辑**：当 `type="multiple"` 时，`value` 和 `defaultValue` 必须是字符串数组。
2. **样式隔离**：在 `outline` 变体下，通过 CSS 选择器处理了中间按钮的边框合并和圆角重置。
3. **无障碍**：完全支持键盘 `Tab` 切换和方向键（在单选模式下）导航，符合 ARIA 最佳实践。
