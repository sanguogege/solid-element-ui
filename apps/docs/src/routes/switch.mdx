## Switch 开关

允许用户在两个互斥状态（开/关）之间切换。

### 导入

```tsx
import { Switch } from "solid-element-ui";
```

### 基础用法

```tsx
<div className="flex flex-col gap-4">
    <Switch label="启用通知" defaultChecked />

    <Switch
        label="深色模式"
        description="自动根据系统设置调整外观。"
        variant="success"
        size="lg"
    />

    <Switch label="不可用选项" disabled />
</div>
```

---

### API 参考

| 属性               | 类型                                 | 默认值      | 说明                     |
| :----------------- | :----------------------------------- | :---------- | :----------------------- |
| **label**          | `string`                             | -           | 开关旁边的文本标签。     |
| **description**    | `string`                             | -           | 标签下方的辅助说明文字。 |
| **size**           | `'sm' \| 'md' \| 'lg'`               | `'md'`      | 开关的尺寸规格。         |
| **variant**        | `'primary' \| 'success' \| 'danger'` | `'primary'` | 选中时的配色方案。       |
| **checked**        | `boolean`                            | -           | 受控选中状态。           |
| **onChange**       | `(val: boolean) => void`             | -           | 状态改变时的回调。       |
| **defaultChecked** | `boolean`                            | -           | 非受控初始选中状态。     |
| **disabled**       | `boolean`                            | `false`     | 是否禁用开关。           |

---

### 特性说明

1. **动画**：利用 Tailwind 的 `transition-all` 和 `data-[checked]` 状态实现滑块平滑移动。
2. **辅助功能**：继承 Kobalte 的原生语义，自动关联 Label 与 Input，支持键盘 `Space` 键操作。
3. **自适应布局**：如果提供了 `description`，组件会自动转为垂直对齐的布局模式以保证排版美观。
