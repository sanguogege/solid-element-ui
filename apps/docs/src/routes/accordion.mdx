# Accordion 折叠面板 (Tailwind v4)

基于 **Kobalte** 与 **Tailwind Variants** 构建的高性能配置式组件。

## 导入

```tsx
import { Accordion } from "solid-element-ui";
```

---

## 基础用法

```tsx
const items = [
    { value: "1", title: "第一项", content: "内容部分" },
    { value: "2", title: "第二项", content: "内容部分" },
];

<Accordion items={items} collapsible />;
```

---

## API 参考

### Accordion Props

| 参数            | 类型              | 默认值  | 说明                             |
| :-------------- | :---------------- | :------ | :------------------------------- |
| **items**       | `AccordionItem[]` | `[]`    | **必填**。渲染面板的数据源。     |
| **multiple**    | `boolean`         | `false` | 是否允许多个面板同时展开。       |
| **collapsible** | `boolean`         | `false` | 是否允许关闭当前已展开的面板。   |
| **class**       | `string`          | -       | 覆盖根容器样式的 Tailwind 类名。 |

### AccordionItem 接口

| 字段         | 类型                    | 说明           |
| :----------- | :---------------------- | :------------- |
| **value**    | `string`                | 唯一标识符。   |
| **title**    | `JSX.Element \| string` | 标题内容。     |
| **content**  | `JSX.Element \| string` | 展开后的内容。 |
| **disabled** | `boolean`               | 是否禁用。     |

## Tailwind CSS v4 样式配置

在 TW v4 中，请直接在你的全局 CSS 文件中添加以下 `@theme` 配置，以支持 Kobalte 的高度过渡动画：

```css
@import "tailwindcss";

@theme {
    @keyframes accordion-down {
        from {
            height: 0;
        }
        to {
            height: var(--kb-accordion-content-height);
        }
    }
    @keyframes accordion-up {
        from {
            height: var(--kb-accordion-content-height);
        }
        to {
            height: 0;
        }
    }

    .animate-accordion-down {
        animation: accordion-down 0.2s ease-out;
    }
    .animate-accordion-up {
        animation: accordion-up 0.2s ease-out;
    }
}
```
