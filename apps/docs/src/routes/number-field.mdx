## NumberField 数字输入框

高度封装的数值输入组件，内置步进器、范围限制和无障碍支持。

### 导入

```tsx
import { NumberField } from "solid-element-ui";
```

### 基础用法

```tsx
<div class="max-w-48">
    <NumberField
        label="购买数量"
        defaultValue={1}
        minValue={1}
        maxValue={10}
        step={1}
    />
</div>
```

### API 参考

| 属性                | 类型                   | 默认值 | 说明                             |
| :------------------ | :--------------------- | :----- | :------------------------------- |
| **label**           | `string`               | -      | 顶部的说明标签。                 |
| **defaultValue**    | `number`               | -      | 初始数值。                       |
| **minValue**        | `number`               | -      | 允许输入的最小值。               |
| **maxValue**        | `number`               | -      | 允许输入的最大值。               |
| **step**            | `number`               | `1`    | 每次点击步进器的增减量。         |
| **format**          | `boolean`              | `true` | 是否根据当前语言环境格式化数字。 |
| **validationState** | `'valid' \| 'invalid'` | -      | 校验状态。                       |

---

### 键盘交互

-   `ArrowUp / ArrowDown`: 增加或减少数值。
-   `Home / End`: 直接跳转至最小值或最大值。
-   `PageUp / PageDown`: 大跨度（通常为 10 倍 step）增减数值。
