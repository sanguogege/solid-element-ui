## Search 搜索框

用于输入查询内容的表单组件，基于 Kobalte TextField 封装。

### 导入

```tsx
import { Search } from "solid-element-ui";
```

### 基础用法

在 MDX 环境下的实例展示：

```tsx
<div className="flex flex-col gap-6 w-full max-w-sm p-4">
    {/* 基础搜索 */}
    <Search placeholder="搜索文档..." />

    {/* 带清除按钮和变体 */}
    <Search
        variant="lg"
        ringColor="danger"
        allowClear
        placeholder="快速跳转..."
    />
</div>
```

---

### API 参考

| 属性            | 类型                    | 默认值      | 说明                                        |
| :-------------- | :---------------------- | :---------- | :------------------------------------------ |
| **size**        | `'sm' \| 'md' \| 'lg'`  | `'md'`      | 搜索框的尺寸。                              |
| **ringColor**   | `'primary' \| 'danger'` | `'primary'` | 聚焦时的光晕颜色。                          |
| **placeholder** | `string`                | `'搜索...'` | 占位提示文字。                              |
| **allowClear**  | `boolean`               | `false`     | 是否允许显示清除按钮（需配合 value 使用）。 |
| **onClear**     | `() => void`            | -           | 点击清除按钮的回调。                        |
| **value**       | `string`                | -           | 受控绑定的值。                              |
| **onChange**    | `(val: string) => void` | -           | 值改变时的回调。                            |

> 继承所有 [Kobalte TextField](https://kobalte.dev/docs/core/components/text-field) 的原始属性（如 `disabled`, `readOnly` 等）。
